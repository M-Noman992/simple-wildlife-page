# Pakistani Wildlife Wonders

A responsive web application designed to showcase rare wildlife species through a clean, nature-inspired interface.

### Key Features

* **Interactive Slideshow**: A JavaScript-powered image slider with start and stop controls.
* **Responsive Design**: Built using Bootstrap 5 to ensure compatibility across mobile, tablet, and desktop devices.
* **Information Gallery**: Detailed sections featuring various species like the Snow Leopard, Markhor, and Indus River Dolphin.
* **Modern Styling**: Custom CSS with smooth transitions, hover effects, and a professional layout.

### Technical Stack

* **Frontend**: HTML5, CSS3
* **Framework**: Bootstrap 5 (CDN)
* **Scripting**: Vanilla JavaScript

### Setup Instructions

1. Clone this repository to your local machine.
2. Ensure you have an `imgs` folder containing the necessary image assets.
3. Open `simple-wildlife-page.html` in any modern web browser to view the site.
